# Codespaces Repository

This repository is configured for GitHub Codespaces development.

## Quick Start

1. Click the "Code" button on GitHub
2. Select the "Codespaces" tab
3. Click "Create codespace on main"
4. Wait for the environment to initialize

## Environment

The devcontainer is configured with:
- Universal Linux development container
- Git and GitHub CLI
- VS Code extensions (Python, C++, PowerShell, GitHub Copilot)

## What's Included

- `.devcontainer/devcontainer.json` - Development container configuration
- `.github/workflows/codespaces.yml` - CI/CD workflow
- `README.md` - This file

## Getting Started

Add your project files and update the configurations as needed for your development needs.
